Om du vill läsa från en fil som inte använder ett standardformat (UTF-8 t.ex.) så behöver du wrappa strömmarna
och explicit sätta standardformatet egnom InputStreamReader och FileInputStream. Detta eftersom FileReader förväntar sig ett standardformat.

T.e.x

BufferedReader br = new BufferedReader( new InputStreamReader( new FileInputStream("test.txt", "UTF-8")));

BufferdWriter writer = new BufferedWriter( new OutputStreamWriter (FileOutputStream("test.txt", true), "UTF-8"))
-----

Generell struktur för ovan är följande:

Reader reader = new Reader( new StreamReader ( new Stream("filnamn","standard format of file content")));

Writer writer = new Writer( new StreamWriter ( new Stream("filnamn","standard format of file content")));
